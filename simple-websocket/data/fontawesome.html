Utilisation de Font Awesome sans connexion Internet

Dans cet exemple, seules les icônes utilisées sont présentes dans le fichier JavaScript qui a été renommé fontawesome-all.min-used-icons-only.js.

Pour enlever les icônes inutilisées, j’ai recherché les numéros de ligne des icônes utilisées et j’ai supprimé les autres manuellement, puis j’ai minifié le fichier avec JSCompress.
